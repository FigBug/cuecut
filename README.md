CueCut uses cuesheets (.cue) files to losslessly cut long mp3 mixes into shorter tracks. 
  
 - Some key features include:
 - Uses pcutmp3 cutting engine
 - Supports gapless playback using LAME tag
 - Supports VBR mp3s
